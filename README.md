### Usage

Usage options as of Version 0.25

| Option        | Description                       |
|---------------|-----------------------------------|
|-a,--archive   |Archive file(s)<br>
  -r,--restore  |Restore file(s) from archive<br>
  -v,--view     |View file(s) in an archive file<br>
  -h,--help     |Show usage information<br>
